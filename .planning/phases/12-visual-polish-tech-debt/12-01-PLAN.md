### Task 2: Add checkmark feedback to CopyButton

**Do:**
1. Edit `src/components/summary/CopyButton.tsx`:
   - Add `import { useState } from 'react';`
   - Add `copied` state and timer to handleClick:
     ```tsx
     export function CopyButton({ onClick, ariaLabel }: CopyButtonProps) {
       const [copied, setCopied] = useState(false);

       function handleClick(e: React.MouseEvent) {
         e.stopPropagation();
         onClick();
         setCopied(true);
         setTimeout(() => setCopied(false), 1500);
       }
     ```
   - Replace the SVG rendering with a conditional:
     - When `copied` is true, show a green checkmark SVG
     - When `copied` is false, show the existing clipboard SVG
   - Checkmark SVG:
     ```tsx
     {copied ? (
       <svg
         xmlns="http://www.w3.org/2000/svg"
         width="16"
         height="16"
         viewBox="0 0 16 16"
         fill="none"
         stroke="currentColor"
         strokeWidth="2"
         strokeLinecap="round"
         strokeLinejoin="round"
         className="text-green-400"
         aria-hidden="true"
       >
         <polyline points="3 8 7 12 13 4" />
       </svg>
     ) : (
       /* existing clipboard SVG unchanged */
     )}
     ```
   - Update button className to reflect copied state:
     ```tsx
     className={`${copied ? 'text-green-400' : 'text-gray-500 hover:text-gray-300'} min-w-10 min-h-10 flex items-center justify-center`}
     ```

**Verify:**
- `npx tsc --noEmit` passes

- `npm run build` succeeds
- Visual: clicking copy shows green checkmark for 1.5s then reverts
